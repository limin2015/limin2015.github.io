1. 我在代码中看过这句：（这两种形式，当memset为1时，都不work，原因看上面的博客《正确使用memset》）

    int resCount_h[PROC_MAX_NUM];
    memset(&(resCount_h[0]), 0, sizeof(int) * PROC_MAX_NUM);

这一句等价于：

    int data[10];  
    memset(data, 0, sizeof(data));    // right
