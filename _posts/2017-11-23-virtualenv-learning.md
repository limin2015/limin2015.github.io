## 是什么？

建立隔离的python工作环境

## 安装和使用

	pip install virtualenv //安装
	virtualenv httpenv     //建立一个虚拟环境
	source httpenv/bin/activate //启动，激活环境
	deactivate  //退出虚拟环境
